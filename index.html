Seletores

Seletores são palavras padrões que utilizamos no CSS para escolher qual elemento vamos escolher para estilizar.

Podemos escolher qualquer seletor, inclusive a Tag HTML.

Tag

Para estilizar usando uma Tag:

nome-da-tag {
        estilo: valor;
}

Class

Para estilizar usando uma class:

.nome-da-class {
        estilo: valor;
}

Todos os elementos

Para mudar o margin e o padding de toda a página:

* {
        margin: valor;
        padding: valor;
}

Duas ou mais class

Este primeiro parágrafo é vermelho:

.paragrafo.vermelho {
        color: red;
}

Já este segundo parágrafo é verde:

.paragrafo.verde {
        color: green;
}

.paragrafo {
        font-weight: 700;
}

Assim estilizamos o peso da fonte dos dois parágrafos ao mesmo tempo, ficando somente a cor diferente.

Estilizando com vírgula

Podemos estilizar dois elementos diferentes com a mesma regra, separando-os com uma vírgula.

.paragrafo, span {
        font-weight: 700;
}

Parágrafo

Span

Estilizando com class filha

Este parágrafo é filho de uma section

.class-pai .class-filha {
          color:green;
}

Estilizando com seletor >

Também podemos estilizar vários elementos filho utilizando o elemento pai seguido do sinal de >

div > p {
        color: green;
}

Estilizando com seletor +

div para exemplificação.

Este parágrafo está localizado no HTML, após uma div.

div + p {
        color: green;
}

Estilizando com atributo

a[target=_blank] {
        color: green;
}

Somente a Tag <a> que tem o atributo target=_blank receberá o estilo.
